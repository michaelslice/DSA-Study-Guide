Goal: Find if a persons age listed in a vector is over 60 

Explanation

1. Create a variable to store the count

2. Iterate through each element in the vector

3. Next extract the persons age from the 11th and 12th element in the vector and convert it to the corresponding integer

4. Add a conditional if statement to increment count if the 11th and 12th element is greater than 60

5. Return count
