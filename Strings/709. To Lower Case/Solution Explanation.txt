Goal: Iterate through a string and replace all uppercase letters with the corresponding lower case letter

Explanation:

1. First create a for loop to iterate through the string

2. Secondly create a nested if statement in the for loop so if the element is uppercase, change the character to lowercase

 for(char &character: s)
    {
        if(std::isupper(character))
        {
            character = std::tolower(character);
        }
    }

3. Finally return the string
