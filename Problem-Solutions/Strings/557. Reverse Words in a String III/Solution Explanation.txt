Goal: Given a string s, reverse the orer of characters in each word within a sentance while still preserving whitespace and the initial order

Explanation:

1. First push back and ' ' in the string

2. Secondly create and string called answer, and str

3. Make a for loop to iterate through string s

4. In the for loop create a nested if statement so if c == ' ' reverse str, and add it to the answer, and then add the space back to ans

Ex:

    for(auto c: s)
    {
        if (c== ' ')
        {
            std::reverse(str.begin(), str.end());
            ans += str;
            ans += ' ';
            str.clear();
        }
        else str.push_back(c);
        
    }

5. After doing so clear the str

6. Finally pop_back the ans, which will remove the last element in ans 

7. Finally return ans
